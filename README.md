## Setting it up locally

 - Clone the repo
 - Copy over .env.example over to .env everywhere
 - Update .env
    - Postgres DB Credentials
 - npm install
 - Start ws server
    - cd apps/ws
    - npm run dev
 - Start http
    - cd apps/http
    - npm run dev
 - Start frontend
    - cd apps/frontend
    - npm run dev
